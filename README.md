# SysAdmin Prep ⚙️

פלטפורמת לימוד אינטראקטיבית להכנה לראיון **Junior System Administrator**
(Windows · Active Directory · Networking · Cloud).

מבוססת על חוברת ההכנה המקורית והורחבה לאפליקציית לימוד מלאה.

## הפעלה

אין צורך בהתקנה, בשרת או בבנייה — פשוט פותחים את הקובץ:

```
index.html
```

בלחיצה כפולה בדפדפן (Chrome / Edge / Firefox). הכול עובד מקומית (גם מ-`file://`).

## מה יש בפנים

- **19 מודולים** — AD+FSMO, DNS, Replication, DNS Flow, Networking, Kerberos,
  Permissions, Backup, VMware, Azure/Entra, Troubleshooting, Event IDs, Ports,
  **Linux (יסודות + תפעול), PowerShell, Windows Infrastructure מתקדם (PKI/DHCP/WSUS/Exchange),
  ראיון התנהגותי (STAR)**, וסיכום.
- **לימוד** — מושגים, טבלאות, שלבי תהליך ותרחישי אבחון מסודרים.
- **כרטיסיות (Flashcards)** — לכל מודול, מצב מעורב על כל החומר, ודירוג "ידעתי / לא ידעתי".
- **קוויזים** — שאלות אמריקאיות עם משוב והסבר; קוויז עם 80%+ מסמן מודול כהושלם.
- **מבחן מדומה מתוזמן** — 10/20/מלא, שאלות אקראיות מכל המודולים, עם שעון וניקוד שיא.
- **חזרה חכמה** — אוסף אוטומטית כל שאלה שטעיתם בה וכל כרטיסייה שסומנה "לא ידעתי", לתרגול ממוקד.
- **טבלאות עזר** — כל הטבלאות במקום אחד לחזרה מהירה לפני הראיון.
- **חיפוש** חופשי בכל התוכן.
- **מעקב התקדמות** + **מצב כהה/בהיר**, נשמרים אוטומטית בדפדפן (localStorage).

## מבנה הפרויקט

```
index.html          נקודת הכניסה
assets/
  styles.css        עיצוב (RTL, dark/light)
  app.js            לוגיקת האפליקציה (vanilla JS)
  data.js           כל תוכן הלימוד — לעריכה/הרחבה כאן
```

## הוספת תוכן

כל התוכן נמצא ב-`assets/data.js` תחת `window.PREP_DATA.modules`.
כל מודול תומך בסקשנים מסוג `list` / `table` / `steps` / `callout`,
ובמערכי `flashcards` ו-`quiz`. הוספת מודול = הוספת אובייקט למערך.

## דרך הצמיחה ל-SaaS אמיתי

הגרסה הנוכחית היא Frontend עצמאי (Client-Side). כדי להפוך לשירות מנוהל:
חשבונות משתמש + שמירת התקדמות בענן (Auth/DB), ניהול תוכן, ותשלומים.
ה-`data.js` מתוכנן כך שניתן להחליפו בקלות ב-API.
